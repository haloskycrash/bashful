# bashful
Minimal ~/.bashrc .  Let's you know who yu are and where you in the console.
To use simply:
1. Copy all the code.
2. Make a backup of your .bashrc
# cp ~/.bashrc ~/.bashrc.backup
3. Next open .bashrc with nano.
# nano ~/.bashrc
4. Delete the code paste bashful code and save.
5. Start feeding commands to your computer.
